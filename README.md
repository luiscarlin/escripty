# escripty

This is a sample multi-module project using Gradle and Liberty

## Run

- Clean build and run

  ```bash
  git clone <this-project>
  cd <this-project>

  cd escripty/build
  ./gradlew clean build libertyStart
  ```
- Open http://localhost:9080/myLibertyApp


> Note: Swagger user/pass = guest/password
